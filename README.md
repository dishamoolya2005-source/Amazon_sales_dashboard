# 📊 Amazon Sales Dashboard

## 📌 Overview

This project is an interactive Amazon Sales Dashboard created to analyze sales performance, profit trends, and customer behavior. It helps in understanding business insights through visual representation.

## 🎯 Objective

The main goal of this project is to:

* Analyze sales across different regions
* Identify profitable categories and sub-categories
* Understand customer purchase behavior
* Track sales and profit trends over time

## 🛠️ Tools & Technologies Used

* Power BI / Excel / Python (whichever you used)
* Data Visualization Techniques
* Data Cleaning & Analysis

## 📂 Dataset

The dataset contains sales-related information such as:

* Sales Amount
* Profit
* Category & Sub-category
* Region & State
* Ship Mode
* Payment Mode
* Return Status

## 📊 Dashboard Features

### 🔹 Sales Analysis

* Sales by Segment (Consumer, Corporate, Home Office)
* Sales by Payment Mode (Cards, COD, Online)

### 🔹 Trend Analysis

* Monthly and Yearly Sales Trends
* Seasonal fluctuations in sales

### 🔹 Category Insights

* Sales by Category (Office Supplies, Technology, Furniture)
* Top 3 profitable sub-categories

### 🔹 Logistics Analysis

* Sales by Ship Mode (Standard Class, First Class, Same Day)

### 🔹 Profit Analysis

* Sales vs Profit per Quarter

### 🔹 Geographic Analysis

* Sales distribution across states (map visualization)

### 🔹 Return Analysis

* Product return status (Accepted vs Returned)

## 📈 Key Insights

* Consumer segment contributes the highest sales
* COD and Online payments are widely used
* Technology category generates high revenue
* Certain sub-categories like Copiers are highly profitable
* Sales show seasonal growth trends

## 📌 Conclusion

This dashboard provides a clear and interactive way to analyze sales data and helps in making data-driven business decisions.
